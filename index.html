Q&A

  1. Defference between Web1,Web2,Web3 :

    • Web 1 used for read-only, this type of Web is just a network information source that has enabled the exchange of other information sources such as encyclopedias/books.

    • In contrast to Web 1, Web 2 also enables writing as well as reading and thus enables social networks that include information sharing.

    • Web 3 is much more advanced and can compare pages while searching and adjust the desired result by user type. Of course also allows reading and writing.


  2. Explanations by example:

    • In Web 1 the search is performed by the search engine displaying sites containing common search words.

    • In Web 3 the search engine will offer the user an autocomplete and the results displayed will be the best match for the user type according to their browsing history and search.
